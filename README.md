# Installation du Front-End

1- Désinstaller Nodejs

2- Installer Nvm https://github.com/coreybutler/nvm-windows

3- Dans un terminal, installer deux version de node grâce à nvm avec le commande nvm install <version>
  
    a- Nous avons besoin de la version 10.0.0 pour l'utilisation du front-end
    
    b- Nous avons besoin de la version 14.16.0 pour l'utilisation du back-end
    
4- Dans le dossier front-end, ouvrir un terminal et utiliser node 10.0.0 grâce à la commande nvm use 10.0.0

5- Nous utilisons maintenant NodeJs dans sa version 10.0.0

6- Toujours dans le dossier front-end, faire la commande npm install

7- Puis npm install node-sass@4.9.3

8- npm start

9- Le serveur front-end est lancé, nous pouvons mettre de coté le terminal et se rendre sur http://localhost:4200/login

# Installation du Back-End

1- Dans le dossier back-end, ouvrir un terminal

2- Changer de version node grâce à la commande nvm use 14.16.0

3- Faire la commande npm install

4- Remplacer le fichier .env.example par un fichier .env en entrant vos informations de connexion à votre DB mongo en suivant le format de .env.example

5- npm start

6- Le serveur back-end est lancé sur le port 3000 et le front-end peut interagir avec.
